## How to Install Kali Terminal 

`$ cd $HOME`

`$ pkg up -y`

`$ pkg install git -y`

`$ git clone https://github.com/Star-Ford/TKH`

`$ cd TKH`

`$ chmod +x *`

`$ bash setup.sh`
